# Liuli Society of America — Multi-page Production Ready Website

This version keeps the homepage layout and style, adds a working menu, includes a Home button, and creates individual pages for each section.

## Pages
- Home
- About
- Approach
- Pathways
- Research
- Community
- Support
- Contact

## Run locally
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```
